# Black Hole Ray Tracer

General Relativity simulation using C++ and SDL2.
Visualizes warping of spacetime (Gravitational Lensing) using Runge-Kutta 4 integration

![Black Hole Simulation](sim_preview.png)

## Features

* **Real-Time Rendering:** Interactive camera controls.
* **Physics Engine:** Custom RK4 solver for light ray geodesics.
* **Visuals:** Accretion disk lensing and grid distortion.
